# README

Experimental project to try out Hotwire Turbo to create a dynamic menu order form plus pricing engine with zero JS and state management.

## Installation

Ruby local version should be set to 3.1.2

`$ bundle install`

`$ rails db:setup`

## Start Instructions

`$ rails s`

`$ rails tailwindcss:watch`
